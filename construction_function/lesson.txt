Функции конструкторы

Как обычная функция выражения только всегда с ней используется 'new' для создания новых однотипных объектов!

1.This возвращает объект
2.в него наваливаются все переменные
3.создается прототип
4.возвращается This

ВЫЗОВ ФУНКЦИИ КОНСТРУКТОРА БЕЗ ОПЕРАТОРА NEW НЕ ИМЕЕТ СМЫСЛА

Всегда юзать строгий режим везде! 'use strict'

У функций конструкторов есть три основных метода вызова без модификации

call - простой вызов
apply - все аргументы кроме зыса передаются массивом

bind - привязывает контекст
